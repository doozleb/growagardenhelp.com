Values

This page provides a comprehensive list of all fruits and their values in Grow A Garden. Use this as a reference to determine which crops are most profitable and to guide your planting strategy.

Calculation of Sheckles Value

Total Value = Base Value × Growth Mutation × (1 + Total Mutation − Number of Environmental Mutations)

  • Base value: Value without any mutations
  • Growth mutation: Gold (20) or Rainbow (50) or none (1)
  • Total mutation: Sum of all multipliers (add them all up)
  • Number of environmental mutations: How many mutations NOT INCLUDING Gold or Rainbow
